# AutoID - Identificador de Marcas de Autos

Este proyecto utiliza inteligencia artificial para identificar la marca de un vehículo a partir de una foto.

## Tecnologías

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Python, FastAPI, PyTorch (Modelo de IA)

## Configuración del Proyecto

### Requisitos Previos

- Node.js & npm
- Python 3.8+

### Instalación y Ejecución del Frontend

1. Navega al directorio del frontend:
   ```bash
   cd auto-detect-magic
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

### Despliegue

Para construir la aplicación para producción:

```bash
npm run build
```
